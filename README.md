# Deep Learning with PyTorch

This repository contains my learning notes, exercises, and PyTorch implementations while studying the book **Deep Learning with PyTorch** by Howard Huang.

I am currently pursuing a PhD in Artificial Intelligence at the University of Pisa, and this repository documents my practical journey in deep learning, neural networks, and PyTorch-based model development.

## Repository Goals

- Learn PyTorch fundamentals
- Implement neural network models from scratch
- Practice tensor operations, autograd, and training loops
- Maintain chapter-wise notes and exercises
- Build a strong foundation for AI research and applied deep learning
